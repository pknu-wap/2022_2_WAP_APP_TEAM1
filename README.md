<h2>Android Tech Stack</h2>

- Minumum SDK 23
- Kotlin
    - Coroutines
- Dagger-Hilt
- Room
- Android Jetpack
    - LiveData
    - Navigation Component
    - SharedPreference
    - DataBinding
- Retrofit2 & OkHttp3 & Moshi
- Glide
- Material Components
- Canhub ImageCropper
- hdodenhof CircleImageView

<h2>Android Architecture</h2>
